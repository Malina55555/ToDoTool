--- Отметить задачу как выполненную ---
Для того, чтобы отметить задачу как сделаннную, нажмите на пустой квадратик рядом с её текстом. Галочка в квадратике сообщает о том, что задача выполнена.
--- Добавление новой задачи ---
Для добавления новой задачи введите её текст в поле "Введите текст новой задачи / текст для её поиска". Если Вам нужно указать крайний срок выполнения задачи, то укажите её в поле "31/12/2023", иначе оставьте поле пустым. Если эта задача является для Вас важной, то поставьте галочку перед надписью "!!!". После этого нажмите кнопку "Сохранить", расположенную на той же строчке, что и поля ввода.
--- Редактирование задачи ---
Для редактирования задачи нажмите кнопку "..." напротив её текста. Перед вами появятся поля, заполненные в соответствии с существующей задачей. Для сохранения изменений нажмите кнопку "Сохранить". Для удаления задачи - "Удалить". Если же Вы не хотите сохранять изменения, нажмите кнопку "Отмена". 
--- Поиск задачи ---
Для поиска нужной задачи введите её текст (или его часть) в поле "Введите текст новой задачи / текст для её поиска". Для поиска задачи с определённой датой выполнения, её также нужно в это поле! (Не в поле "31/12/2023"!) После заполнения поля нажмите кнопку "Найти".
--- Фильтрация запросов ---
Для фильтрации запросов по определённому признаку нажмите соответствующую кнопку:
"Все" - выведет все существующие задачи.
"Завершённые" - выведет все выполненные задачи.
"Незавершённые" - выведет все невыполненные задачи.
"Просроченные" - выведет невыполненные задачи с истёкшим сроком выполнения. Даты таких задач помечены красным.
"Важные" - выведет все задачи с пометкой важные ("!!!").
--- Удаление задачи ---
Удаление задачи осуществляется в окне её редактирования по кнопке "Удалить".
